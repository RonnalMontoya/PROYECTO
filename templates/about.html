ℹ️ Acerca del Sistema

Este sistema de inventario fue desarrollado como parte del curso de Desarrollo Web con Flask. Su objetivo es demostrar cómo integrar una aplicación Flask con una base de datos MySQL, permitiendo realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar) sobre la tabla producto.

🛠 Funcionalidades principales:

  • ✅ Listar todos los productos registrados
  • ✅ Buscar productos por nombre
  • ✅ Añadir un nuevo producto al inventario
  • ✅ Editar la información de un producto existente
  • ✅ Eliminar productos de la base de datos

👨‍💻 Tecnologías utilizadas:

  • ⚡ Python 3 con el microframework Flask
  • 🗄 MySQL gestionado desde phpMyAdmin
  • 🎨 HTML5, Bootstrap 5 para la interfaz
Nota: Este sistema es un prototipo académico y puede adaptarse a distintos escenarios reales de gestión de inventarios.
